# Intelligent Legal Case Analysis System

## Overview
This project is an NLP-based system designed to analyze legal case documents and identify similar judgments. The system processes legal texts and applies Natural Language Processing techniques to extract meaningful insights and relationships between cases.

The application is implemented in Python and deployed as an interactive web app using Streamlit.

---

## Dataset
The dataset used in this project is:

**Legal Dataset: SC Judgments India (1950–2024)**  
Source: Kaggle  
Collected from Indian Kanoon with over **98% coverage of Supreme Court judgments**.

---

## Concepts Used

The project applies several NLP and text analysis techniques:

- Text preprocessing and cleaning
- TF-IDF Vectorization
- Cosine Similarity for document comparison
- Named Entity Recognition (NER)
- Keyword Co-occurrence Matrix

These techniques help in identifying relationships between legal documents and extracting meaningful patterns from case texts.

---

## Features

- Upload and analyze legal case documents
- Identify similar judgments using text similarity
- Visualize important legal terms using word clouds
- Generate keyword co-occurrence heatmaps
- Interactive analytics dashboard for exploring legal data

## Purpose of the Project and outcomes:

The goal of this project is to demonstrate how Natural Language Processing techniques can be applied to legal text analysis. By combining text similarity, entity recognition, and visualization methods, the system helps in understanding relationships between legal cases and extracting useful insights from large legal datasets.
